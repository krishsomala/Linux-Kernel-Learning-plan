Exercise - Build ARM Kernel for Raspberry Pi:
1. Setup Cross-Compiler:
sudo apt install gcc-aarch64-linux-gnu (for 64-bit RPi)
git clone --depth=1 https://github.com/raspberrypi/linux

2. Configure Kernel:
cd linux
KERNEL=kernel8
make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- bcm2711_defconfig
make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- menuconfig
Enable: CONFIG_KGDB=y, CONFIG_KGDB_SERIAL_CONSOLE=y

3. Build:
make -j$(nproc) ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- Image modules dtbs

4. Deploy to RPi:
Copy Image to /boot on SD card
Copy .dtb files to /boot
Copy modules: make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- INSTALL_MOD_PATH=/mnt/ext4 modules_install

5. Setup KGDB:
Add to /boot/cmdline.txt: kgdboc=ttyAMA0,115200 kgdbwait
Connect serial cable, boot RPi, connect from host: aarch64-linux-gnu-gdb vmlinux
(gdb) target remote /dev/ttyUSB0

6. Write Simple GPIO Driver:
Create module to control RPi GPIO pins
Use gpiolib APIs: gpio_request(), gpio_direction_output()
Test with LED connected to GPIO pin
Debug using KGDB: set breakpoints in your driver's probe function
